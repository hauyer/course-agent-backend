# 更新日志

## 1.1.0 - 2026-07-15

### Added

- Agent 回答的真实课程资料引用与历史恢复。
- 可预览、确认和重新生成的多课程综合规划。
- 带任务状态、版本保护和资料证据的课程知识图谱。
- 用户头像上传，并在侧栏和课程问答中保持一致。
- 多文件上传确认队列、逐文件进度和跨页面上传状态。
- 显式 ChromaDB cosine collection、真实余弦相似度及安全重建脚本。
- Alembic 1.0 基线与 1.1 增量数据库迁移。

### Changed

- Agent 引用与执行过程分开持久化。
- 单课程计划保持兼容，学习计划页面新增综合规划入口。
- 桌面程序直接进入登录页，数据库配置由本机 `.env` 或旧版加密配置提供。
- 前端业务页、Markdown/KaTeX 和图谱按模块懒加载。

### Fixed

- citations 固定为空或与 Agent trace 混用。
- 课程、用户和资料检索边界不统一的问题。
- 弹窗内容被裁切、聊天输入消失和多文件选择不可确认等界面问题。
- 桌面端误复用旧版 8000 端口后端，导致知识图谱、综合规划返回 `Not Found` 的问题。
- 新余弦集合尚未建立时检索为空，以及异常分数显示为 `NaN%` 的问题。
- 上传完成进度条长期残留、原生删除课程弹窗、综合规划左右拥挤排版等交互问题。
- 长回答期间强制滚动造成回复看似自动收起，以及头像选择区文字重叠的问题。
- 本机加密密钥变化后旧 LLM 密钥无法解密时缺少明确提示的问题。
