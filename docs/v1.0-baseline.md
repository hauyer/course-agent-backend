## 安全回滚

仅查看或运行原稳定版本时：

```powershell
git switch main
```

需要从本地恢复标签创建独立修复分支时：

```powershell
git switch -c restore/v1.0.0 v1.0.0-local-backup
```

以上操作不会删除 `release/1.1.0` 分支。禁止使用 `git reset --hard`、`git clean -fd`、`git checkout -- .` 或任何会丢弃本地内容的命令回滚。

## 禁止提交的本地文件

以下内容仅保留在开发电脑或用户数据目录，不得进入 Git：

- `.env` 和任何真实密钥、数据库密码、第三方 API Key；
- `uploads/` 用户课程资料；
- `chroma_db/` 本地向量索引；
- `.venv/`、`node_modules/` 依赖目录；
- `build/`、`dist/`、`release/` 等构建产物；
- MySQL 数据文件、应用备份包、运行日志和 Electron 用户数据。
